# 🏠 House Price Prediction System

## 📌 Overview
This project predicts house prices using machine learning based on features like area, bedrooms, and location.

## 🚀 Features
- ML-based prediction
- Data visualization
- Simple input system

## 🛠 Tech Stack
Python, Pandas, Scikit-learn, Matplotlib

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Train model:
   python train.py

3. Run prediction:
   python app.py

## 📊 Output
Predicts house price based on user input.
